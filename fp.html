liqen/environment

What is this about?

Problematic

  • Not enough or too much complicated tools for citizenship data (API, JSON?)

  • Isolated groups of citizens working with data

  • There is not enough data devolution [not just products... show me your data! Please ;) ]

Solution

We want to make visible the efforts of numerous citizens who are using and producing open data everyday. We believe than we can cooperate creating new tools to allow that people working on the new field of data citizenship can collaborate with each other in a better way.

So... What is liQen?

liQen is a set of tools aims to...

  • Collect

  • Deduplicate

  • Share
